## Compatible with any ESP32 microcontroller, without a screen, supporting native LN invoices and static LNURLPay links.

> <a href="https://youtu.be/FeoIwTjv3YM">Video tutorial</a>

> Join us! <a href="https://t.me/makerbits">t.me/makerbits</a>

### Install instructions
- Install <a href="https://www.arduino.cc/en/software">Arduino IDE 1.8.19</a>
- Install ESP32 boards, using <a href="https://docs.espressif.com/projects/arduino-esp32/en/latest/installing.html#installing-using-boards-manager">boards manager</a>

- Download this repo
- Copy these <a href="libraries">libraries</a> into your Arduino install "libraries" folder
- Open this <a href="bitcoinSwitch.ino">bitcoinSwitch.ino</a> file in the Arduino IDE
- Select the correct ESP32 board from tools>board
- Upload to device
- Default password is "ToTheMoon"

### 4Amounts3Outputs.ino
- Script for 4 static amounts and 3 output-pins
- Amounts and output-pins set in the Arduino-IDE, not in the Portal
- NO M5Stack support!!
- NO Portal support!!
